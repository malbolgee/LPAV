)

Econtra a menor distância entre SRC e todos os outros nós do grafo.

Parâmetros
graphÉ um ponteiro para um grafo.
distÉ um vetor de distâncias.
mwidthÉ a largura de cada membro em dist.
srcÉ o nó de origem do cálculo de distâncias.
bdistÉ a distância de src para ele mesmo.
infcÉ uma constante que representa uma distância inalcançável entre quaisquer dois nós
compareÉ uma função de comparação usada para comprar dois elementos de dist.
assignÉ função para fazer atribuição de valores em dist.
operationÉ uma função que realiza operações aritméticas entre dois elementos de dist ou entre um elemento de dist e um número.
Aviso
bdist e infc precisam ser do mesmo tipo de dist.
Retorna
As distâncias estão em dist.
